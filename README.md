# RangeLine

## Overview
**RangeLine** is a Python-based golf shot tracker that allows users to select a club, record shot distances, and view performance trends over time. The program saves data locally and uses basic analytics to help golfers better understand their consistency and distance control.

## Features (Completed ✅ & In Progress 🔄)
- ✅ Choose a golf club to track.
- ✅ Enter any number of shot distances for the selected club.
- ✅ Option to remove the most recently entered shots (in case of mistyped inputs).
- ✅ Save shot data to a JSON file for persistence.
- ✅ Calculate and display the average distance of the past 5 shots (if available).
- ✅ Display a trend line of the last 15 shots using Matplotlib for easy visualization (if available).
- 🔄 Currently optimizing code structure, improving modularity, and adding error handling.

## Tech Stack
- **Language:** Python 3.11  
- **Libraries:** `json`, `matplotlib`, `collections`
